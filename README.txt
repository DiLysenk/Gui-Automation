Скрипт для автоматизации действий в GUI приложениий

1. Создать папку buttons и в эту папку сохранить по порядку скриншоты объектов с которыми надо взаимодействовать
    1.1 скриншоты нумируются по правилу 01, 02....11 ... 99 и тд
    1.2 комментарии к скриншотам должны быть подписаны на анг языке
2. параметр '!' указывает на задержку перед действием, задержку писать 2мя знаками после "!" (01 или 02, 33, 99)
3. параметр 'rr' указывает на ПКМ
4. параметр 'dd' указывает на двойной клик левой кнопкой
