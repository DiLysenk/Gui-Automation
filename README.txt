Скрипт для автоматизации действий в GUI приложениий

Установка:
    1. Установить Git.
    2. Скачать проект Gui-Automation на машину с  помощью команды в
    (git clone https://github.com/DiLysenk/Gui-Automation).
    3. Установить интерпретатор python
    4. Установить библиотеки pillow, pyautogui,numpy, opencv-python с помощью команды в консоли "pip install".
    5. Создать папку buttons_bla в корне проекта и в эту папку сохранить по порядку скриншоты объектов с которыми надо
     взаимодействовать.
    6. Скриншоты нумеруются по правилу 001, 002...011...199 и т.д.
    7. Комментарии к скриншотам должны быть подписаны на латинице языке
    8. Параметр '!' указывает на задержку перед действием, задержку писать 2мя знаками после "!" (01 или 02, 33, 99)
    9. Параметр 'rr' указывает на ПКМ
    10.  Параметр 'dd' указывает на двойной клик левой кнопкой
